✅ Task C1: User & Access Hardening (8 Marks)
STEP 1: Create New User cloudadmin
sudo adduser cloudadmin

STEP 2: Add User to Sudo Group
sudo usermod -aG sudo cloudadmin
Verify: groups cloudadmin(cmd)
You should see:
    cloudadmin : cloudadmin sudo


STEP 3: Test Sudo Access (IMPORTANT)
Switch user: su - cloudadmin
Test sudo: sudo whoami
Output: root

STEP 4: Disable Direct Root SSH Login
Edit SSH config: sudo nano /etc/ssh/sshd_config
Find: #PermitRootLogin prohibit-password
Change to:  PermitRootLogin no
Restart SSH: sudo systemctl restart ssh


✅ Task C2: Log Backup Script (6 Marks)
🎯 Objective
Archive /var/log
Use timestamped .tar.gz
Store in /home/ubuntu/logbackup

STEP 1: Create Backup Directory
mkdir -p /home/ubuntu/logbackup

STEP 2: Create Script File
nano /home/ubuntu/logbackup.sh

STEP 3: Paste Script Content
#!/bin/bash

BACKUP_DIR="/home/ubuntu/logbackup"
TIMESTAMP=$(date +"%Y-%m-%d_%H-%M-%S")
BACKUP_FILE="logbackup_$TIMESTAMP.tar.gz"

tar -czf $BACKUP_DIR/$BACKUP_FILE /var/log

STEP 4: Make Script Executable
chmod +x /home/ubuntu/logbackup.sh

STEP 5: Run Script Manually
sudo /home/ubuntu/logbackup.sh
Verify: ls -lh /home/ubuntu/logbackup
    cat /home/ubuntu/logbackup.sh

✅ Task C3: Scheduled Automation (Cron) (6 Marks)
🎯 Objective
    Run backup daily at 1:30 AM

STEP 1: Edit Crontab
crontab -e

STEP 2: Add Cron Entry
30 1 * * * /home/ubuntu/logbackup.sh

STEP 3: Verify Cron Job
crontab -l
